Please use following structure when inserting the algorithm
1. The algorithm should be in javascript language.
2. Base case and the recursive steps should be included inside the "function f(n){}" phase.
3. If there are multiple inputs, the phrase should be "function f(n,k,....){}". 4. Variable values should be included in the second input space provided. One or more variables should be included in the correct order as mentioned in the function.

EXAMPLE CODE

function f(n){if (n<=1) {return 1;}else{return n*f(n-1);}}
